# Windows.Devices.Midi2

Temporary NuGet package for Windows.Devices.Midi2 APIs.

- Main site with SDK documentation: https://aka.ms/midi
- Discord for discussion: https://aka.ms/mididiscord
- Repo : https://aka.ms/midirepo

When using this NuGet package, note there is no pre-built .NET projection, as this is also how it will work in-box. Instead, your project must use the latest cswinrt to create projections from the .winmd files
